== Description ==

Enable user to share their invoices with as many email address as they like, all from the order review and view order pages. Simply install the plugin, customize your messages and that's it.

Features:

* Simple installation makes getting started easy.
* Customize all form field values, error messages and the page/position the form is displayed on.
* Edit the email template under WooCommerce's email settings from full integration with WooCommerce.
* Validates users to prevent unauthorized usage.
* Easily customize email subject
* Uses default WooCommerce Order Template

[Plugin's Official Documentation and Support Page](http://www.wpbackoffice.com/plugins/woocommerce-invoice-sharing)

== Installation ==

Automatic WordPress Installation

1. Log-in to your WordPress Site
2. Under the plugin sidebar tab, click ‘Add New’
3. Search for ‘WooCommerce Invoice Sharing'
4. Install and Activate the Plugin
5. Visit WooCommerce -> Invoice Sharing to adjust plugin settings
6. Visit WooCommerce -> Settings -> Emails -> Share Order to adjust email settings

Manual Installation

1. Download the latest version of the plugin from WooCommerce Incremental 
2. Product Quantities wordpress page.
3. Uncompress the file
4. Upload the uncompressed directory to ‘/wp-content/plugins/’ via FTP
5. Active the plugin from your wordpress backend ‘Plugins -> Installed Plugins’
6. Visit WooCommerce -> Invoice Sharing to adjust plugin settings
7. Visit WooCommerce -> Settings -> Emails -> Share Order to adjust email settings
